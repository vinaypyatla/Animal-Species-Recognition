# Animal Species Recognition using Deep Learning

## Project Overview
This project classifies animal images into 10 species using Deep Learning models:
- Custom CNN
- MobileNetV2 (Transfer Learning)

The goal is to compare both models and identify which performs better for animal classification.

---

## Dataset
Dataset contains 10 animal classes:
- Dog
- Horse
- Elephant
- Butterfly
- Chicken
- Cat
- Cow
- Sheep
- Spider
- Squirrel

Total images: 26,179

Dataset source: Kaggle Animals-10 Dataset

---

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
- Kaggle Notebook
- Deep Learning

---

## Model Performance

### Custom CNN
- Accuracy: 67.64%
- Loss: 0.9572

### MobileNetV2
- Accuracy: 95.39%
- Loss: 0.1491

---

## Best Model
MobileNetV2 performed significantly better than Custom CNN.
